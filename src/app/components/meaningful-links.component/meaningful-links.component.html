Regla 8: Enlaces Significativos

Los enlaces deben tener textos descriptivos que permitan entender su propósito sin necesidad de contexto adicional. Esto es crucial para usuarios de lectores de pantalla.

❌ Ejemplos Incorrectos

{{ example.title }}

❌ Incorrecto

{{ example.badText }}

{{ example.problem }}

✅ Correcto

{{ example.goodText }}

{{ example.explanation }}

}

💡 Mejores Prácticas

📝 Sé Específico

Usa textos que describan exactamente a dónde lleva el enlace o qué acción realizará.

Ejemplo: "Descargar informe trimestral Q3 2024 (PDF)"

🎯 Incluye Formato

Cuando el enlace lleva a un archivo específico, indica el formato y tamaño.

Ejemplo: "Ver presentación (PowerPoint, 5.2 MB)"

🔗 Evita Repetición

No uses el mismo texto para enlaces que van a diferentes destinos.

Mal: "Leer más" (varias veces)
Bien: "Leer más sobre tema X", "Leer más sobre tema Y"

📱 Funciona sin Estilos

El enlace debe ser comprensible incluso sin CSS ni imágenes.

Ejemplo: "Contactar con soporte técnico" en lugar de solo un icono
