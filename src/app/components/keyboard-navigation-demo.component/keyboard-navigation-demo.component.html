Regla 4: Navegación por Teclado

Todos los elementos interactivos deben ser operables mediante teclado con orden lógico de tabulación.

1. Enlaces de Salto (Skip Links)

Saltar al contenido principal Saltar a navegación Saltar a formularios

✅ Implementación Correcta:

  • Enlaces visibles solo al recibir foco
  • Posicionamiento absoluto con top negativo
  • z-index alto para prioridad
  • Transición suave al recibir foco

2. Orden de Tabulación Lógico

Contenido Principal

El orden de tabulación sigue la estructura visual del documento.

✅ Orden Correcto:

  1. Navegación → Contenido principal → Botones de acción
  2. Respetar la jerarquía visual
  3. tabindex="0" para elementos interactivos
  4. tabindex="-1" para elementos programáticamente enfocados

3. Indicadores de Foco Visibles

Enlace Estándar

✅ Estilos de Foco:

  • outline: 2px solid con alto contraste
  • outline-offset: 2px para separación
  • Colores que cumplen ratio 4.5:1
  • Consistente entre todos los elementos

5. Atajos de Teclado Comunes

Tecla Función Ejemplo
Tab Navegar al siguiente elemento Moverse entre campos
Shift + Tab Navegar al anterior elemento Retroceder en navegación
Enter Activar botones/enlaces Presionar botones
Space Activar botones/checkbox Marcar opciones
Esc Cerrar modales/diálogos Salir de ventanas
Arrow Keys Navegar en listas/menús Moverse entre opciones
